# ReimuruABT Rebuild

This package is the clean rebuild starter for your Railway app.

## What is included
- Premium landing page
- Login + sign up UI shells
- Dashboard
- Upload + analyze flow
- Results page
- History page
- Flashcards page
- Quiz page
- Pricing page
- Settings page
- Express backend with `/api/analyze`
- OpenAI support with a fallback mode

## Quick deploy
1. Upload these files to the **root** of a new GitHub repo.
2. Deploy that repo to Railway.
3. Add `OPENAI_API_KEY` in Railway Variables.
4. Generate a public domain in Railway networking.

## Important note
This rebuild gives you a polished MVP foundation.
Real auth, database-backed history, Stripe billing, and credit accounting per user are the next implementation step.
